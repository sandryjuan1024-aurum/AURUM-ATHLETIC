# 🏆 Aurum Athletic - Landing Page

Bienvenido al repositorio oficial de **Aurum Athletic**, una landing page premium diseñada para atletas de alto rendimiento enfocados en la hipertrofia, fuerza y optimización del entrenamiento.

## 🛠️ Tecnologías Utilizadas
* **HTML5** para la estructura de la página.
* **CSS3** para el diseño visual, diseño adaptativo (responsive) y la estética Dark Mode con detalles dorados.
* **JavaScript (JS Vanilla)** para la lógica de validación interactiva y la calculadora motivacional de series efectivas.

## 🚀 Funcionalidades
* **Validación de Formularios interactiva:** El script intercepta el registro al club, calcula un consejo de volumen de entrenamiento de alta intensidad según el objetivo del usuario y despliega alertas dinámicas en la pantalla (sin usar las molestas ventanas de alerta tradicionales del navegador).
* **Diseño 100% Responsivo:** Adaptado perfectamente para celulares y computadoras.
